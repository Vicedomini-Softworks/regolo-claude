What Actually Happens

Your brain doesn't care about API formats. It cares about Claude answering questions. We handle the translation so you don't have to.

1
Claude Code → POST /v1/messages
Anthropic format (tools, tool_use, system)
→
2
regolo proxy translates
• tools ↔ tools
• tool_use → tool_calls
• system → message
→
3
api.regolo.ai → POST /v1/chat/completions
OpenAI format (tool_calls, role: tool)
→
4
regolo proxy → type: message
Response translated back to Anthropic format

You get Claude Code. We handle the API gymnastics.
